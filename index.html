What is Sveltekit?

SvelteKit is a framework surrounding Svelte, a way of writing HTMl, CSS, JavaScript and components. SvelteKit is essentially a toolkit that brings together Svelte files and compiles them to build a website.

Why should you use SvelteKit?

  1. It's cool
  2. It's faster than React
  3. It has a syntax that is closer to HTML, instead of Java/TypeScript React
  4. It's widely supported
  5. It's cool.
